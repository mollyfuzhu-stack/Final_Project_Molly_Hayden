# Final_Project_Molly_Hayden

The data we have is from an exposure test with the chemical OxyQ and embryonic zebrafish with the genotype wild type (WT) or knockout for cyp 3A65 (KO3A65). Zebrafish are exposed for 5 days to the concentrations 0.25 μm ,0.5 μm ,2 μm ,8 μm ,16 μm plus a control. The endpoints collected from this exposure are survival, hatching, abnormalities frequency, and behaviour such as total distance moved, maximum velocity and time spent at the edge. We plan to do some pairwise analysis between each dose for endpoints survival and abnormalities and behavioural endpoints such total distance moved and maximum velocity. We also plan to do a multivariate analysis to determine if there is relationship between the concentration and endpoints survival, hatch, and abnormalities. We would like to determine if this relationship differs between genotypes. The data can be found in the subfolder data_set.
