# Final_Project_Molly_Hayden

The data we have is from an exposure test with the chemical OxyQ and embryonic zebrafish with the genotype wild type (WT) or knockout for cyp 3A65 (KO3A65). The knockout genotype has an trunkated gene copy of the enzyme cytochrome P450 3A65 which is a part of the chemical defence system. This enzyme is known to have activity with OxyQ as a ligand. The intention of this project is to determine if the enzyme bioactivates the compound and causes its toxicity or bioinactivates it to reduce toxicity. Zebrafish are exposed for 5 days to the concentrations 0.25 μm, 0.5 μm, 2 μm, 8 μm, 16 μm plus a 0 μm control. The endpoints collected from this exposure are survival, hatching, abnormalities frequency, and behaviour such as total distance moved, maximum velocity and time spent at the edge. We plan to do some pairwise analysis between each dose for endpoints survival and abnormalities and behavioural endpoints such total distance moved and maximum velocity. We also plan to do a multivariate analysis to determine if there is relationship between the predictors: day post fertilization (dpf), concentration (dose) and genotype, and response variables: survival (x_dead), hatching from the egg (hatched), and abnormalities. We are most interested if the relationship differs between genotypes and if an interaction exists between dose and genotype. The data can be found in the subfolder data_set.
