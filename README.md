# Final_Project_Molly_Hayden

The data set we are wokring on is from an acclimation exposurure with Killi fish. Killi fish were raised in two conditions, 10 and 24 degrees and in salinity conditions of 11 ppt or 28 ppt for three weeks. Fish were then euthanizes and sampled for gill tisuses to test mitochocondtial enzymes at the two exposures. We plan to do a mulitvariate analaysis to determine how the differnt acclimation conditions impact enzyme activity and body condition. The predictors are temperature, salinity acclimation, temperautre assay, and body condition. The response variables are COX activity, CS activity, LDH activity. 

The data set used for this analysis is in the folder Data_ser and the code for analysis is in the folder Analysis.
